# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a person is likely to have diabetes based on various medical attributes using Machine Learning. A Logistic Regression model is trained on the Pima Indians Diabetes Dataset to classify patients as diabetic or non-diabetic.

This project was completed as part of my **InternPe Machine Learning Internship**.

---

## 🎯 Objective
To build a machine learning model that can predict diabetes based on patient health data and evaluate its performance.

---

## 📂 Dataset
The dataset contains medical information such as:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age
- Outcome (0 = Non-Diabetic, 1 = Diabetic)

---

## 🛠️ Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Scikit-learn

---

## 🤖 Machine Learning Model

**Algorithm Used:**
- Logistic Regression

---

## 📊 Project Workflow

1. Import required libraries
2. Load the diabetes dataset
3. Explore the dataset
4. Split features and target variable
5. Divide data into training and testing sets
6. Train the Logistic Regression model
7. Evaluate model accuracy
8. Predict diabetes for new input data

---

## 📈 Results

- Successfully trained the Logistic Regression model.
- Evaluated the model using the testing dataset.
- Generated predictions for new patient data.
- Achieved good prediction accuracy on the dataset.

> **Note:** The exact accuracy may vary depending on the train-test split.
